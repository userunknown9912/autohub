# BRG AutoHub - Japan Car Export

BRG AutoHub is a premier Japan Car Export specialist platform dedicated to providing top-tier, auction-grade vehicles directly to the Caribbean market.

## Overview

This project is a static website built with HTML, CSS, and Vanilla JavaScript, designed to serve customers looking to import high-quality, right-hand drive Japanese vehicles to countries like Jamaica, Trinidad & Tobago, Barbados, Guyana, and the Bahamas.

## Features

- **Home Page**: Overview of services, statistics, and top export destinations.
- **Island Favorites**: A showcase of the most popular vehicles imported to the Caribbean.
- **How It Works**: A step-by-step guide explaining the direct auction access, strict inspection protocol, and shipping process.
- **Shipping**: Detailed information on RoRo and container shipping schedules directly to major Caribbean ports.
- **About Us & Contact**: Information about BRG AutoHub's headquarters in Tokyo and direct communication lines (WhatsApp, email, and social networks).

## Tech Stack

- HTML5
- CSS3 (Vanilla CSS, custom properties, responsive design)
- JavaScript (Vanilla JS for interactive components)
- Font Awesome (Icons)

## Setup and Usage

1. Clone or download the repository.
2. Open `index.html` in your preferred web browser to view the website locally.
3. For development, you can use any local server (like Live Server in VS Code) to serve the files.
